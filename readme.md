# рсс постер

## суть
парсинг рсс ленты и постинг в телеграм канал через бота с отложенными сообщениями. ~~наговнокожено~~ навайбкожено за 40 минут так что качество кода соответствующее. 

## установка
1. pip install python-telegram-bot feedparser
2. pip install "python-telegram-bot[job-queue]"
3. создайте бота, переименуйте config-sample.py в config.py и заполните все что нужно. если бот не отзывается на /start значит вы не добавили в конфиг себя как администратора
4. добавьте бота как администратора в требуемый канал
проверено на python 3.12.0
## запуск
python main.py
